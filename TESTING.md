# 🧪 Guia de Testes - N-INFO-CELL\n\n## Setup Rápido para Testes\n\n### 1️⃣ Instalar Dependências\n\n```bash\n# Backend\ncd backend\nnpm install\n\n# Frontend (em outro terminal)\ncd frontend\nnpm install\n```\n\n### 2️⃣ Configurar Banco de Dados\n\n```bash\n# Criar banco\npsql -U postgres -c \"CREATE DATABASE n_info_cell;\"\n\n# Importar schema\npsql -U postgres -d n_info_cell -f database/init.sql\n\n# Verificar\npsql -U postgres -d n_info_cell -c \"\\dt\"\n```\n\n### 3️⃣ Configurar Variáveis de Ambiente\n\n**Backend** (`.env`):\n```\nDB_HOST=localhost\nDB_PORT=5432\nDB_NAME=n_info_cell\nDB_USER=postgres\nDB_PASSWORD=postgres\nPORT=3000\nNODE_ENV=development\nJWT_SECRET=test_secret_key_12345\nJWT_EXPIRY=7d\nCORS_ORIGIN=*\n```\n\n---\n\n## ▶️ Iniciar o Sistema\n\n### Terminal 1 - Backend\n```bash\ncd backend\nnpm run dev\n```\n✅ Você verá: `🚀 N-INFO-CELL Backend rodando na porta 3000`\n\n### Terminal 2 - Frontend\n```bash\ncd frontend\nnpm start\n```\n✅ Escolha:\n- `w` para Web\n- `a` para Android\n- `i` para iOS\n\n---\n\n## 🔧 Testar API com cURL\n\n### 1. **Registrar Usuário**\n```bash\ncurl -X POST package-lock.json\nnpm install\n```\n\n---\n\n## 📊 Testar com Postman\n\n1.
